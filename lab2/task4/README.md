## Запуск проекта
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/kira-ko/algs_labs.git
   ```
2. Перейдите в папку с проектом:
   ```bash
   cd algs_labs/lab2/task4
   ```
3. Запустите программу:
   ```bash
   python src/task4.py
   ```

4. Запуск тестов:
   ```bash
   python tests/test_task4.py
   ```
